# 💰 Sistema Bancário em Java

Projeto simples de um sistema bancário feito em Java para praticar lógica de programação e Programação Orientada a Objetos (POO).

## 🚀 Funcionalidades
- Ver saldo
- Depositar dinheiro
- Sacar dinheiro
- Validação de saldo insuficiente

## 🛠 Tecnologias
- Java
- Git e GitHub

## 📚 Aprendizados
- Classes e objetos
- Métodos
- Estruturas de decisão
- Entrada de dados com Scanner

## ▶️ Como executar
1. Rodar a classe Main.java
2. Interagir com o menu no terminal
